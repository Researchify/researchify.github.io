Researchify

An open source lightweight CMS enabling streamlined webpage development for academic research groups.

Installation

Use the package manager npm to install all required components.

# install api
cd api
npm install

# install client
cd client
npm install

To build Docker images in a single line using docker-compose:

# in root project directory
docker-compose build

Usage

Run the api and client using npm.

# run api
cd api
npm run start

# run client
cd client
npm run start

Or via docker-compose:

# in root project directory
docker-compose up

Documentation Site

In the api folder, run:

 ./node_modules/.bin/jsdoc -c ./jsdoc_config.json --readme ./README.md -d ../docs
